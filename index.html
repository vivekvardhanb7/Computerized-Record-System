What is CRS ?

CRS - Computerised Record System

Computerised Record System (CRS) is a website designed to create a labrecord in a computer without any problems. An editor will be provided in this so that the user can type the information that is normally written in the lab subject's record sheets. By putting the lab records online, we can save time and detect plagiarism by comparing the record information submitted by the user and generating a report.



Features

Restriction of copy,paste

Increasing the hands on practise of typing

Record Data Storing

Text Editor to store the Record data
